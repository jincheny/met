## met

mock-express-template

## 介绍
基于 express-generator 二次开发的前端 mock 框架。

## 安装使用

- 获取项目代码

```bash
git clone https://github.com/jincheny/met.git
```

- 安装依赖

```bash
cd met

pnpm install --registry=https://registry.npmmirror.com

```

- 运行

```bash
pnpm run dev

```

浏览器访问 http://127.0.0.1:3000
